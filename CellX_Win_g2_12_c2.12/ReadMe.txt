Release of CellX for the platfrom Windows:

CellX Version 2.12 to be used with CellXGUI version 2.12

Installation of the Matlab MCR 9.0.1 (Matlab 2016a) is required. The MCR can be downloaded from here:
https://ch.mathworks.com/de/products/compiler/matlab-runtime.html

Was tested on platform Windows 7 and 10.
